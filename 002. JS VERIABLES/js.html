JAVASCRIPT VARIABLES (O'zgaruvchilar). (2-DARS)
O'zgaruvchi shunchaki qiymatning egasidir, masalan, elementni ushlab turgan quti.

JS VARIABLES (info)
Oddiy so'zlar bilan aytganda, o'zgaruvchi bu - biron-bir qiymat uchun container (quti) hisoblanadi.
  • O'zgaruvchilarga misollar: - myNumber, my_number, list1, $item, _name, a, b, $, _
    • Quyidagi rasmda to'g'ri o'zgaruvchilarga misollar keltirilgan.
  • O'zgaruvchilarning noto'g'ri misoli: - 1number, my-number, if, let, function va h.k
    • Quyidagi rasmda noto'g'ri o'zgaruvchilarga misollar keltirilgan.
    "JavaScript identifikatorlari katta-kichik harflarga sezgir."
    Masalan: Ism va ism. bu ikki xil o'zgaruvchidir.
JS VARIABLES (JavaScript o'zgaruvchilarini e'lon qilish usullari)
JavaScript-da o'zgaruvchilarni e'lon qilishning 3 ta usuli bor
Bularni birma-bir ko'rib chiqamiz:
1. let
let bu - O'zgaruvchan qiymat.
Quyidagi holat esa xatolikni chiqarib keltiradi!
  • let kalit so'zi orqali e'lon qilingan o'zgaruvchi qiymatini keyinchalik boshqa qiymatga ham o'zgartirib ishlatishimiz mumkin.
  • let kalit so'zi orqali o'zgaruvchini biron-bir blok ichida e'lon qiladigan bo'lsak uni faqat o'sha blok ichida ishlatishimiz mumkin bo'ladi.
2. const
const bu - O'zgarmas qiymat.
  • const kalit so'zi orqali e'lon qilingan o'zgaruvchi doim boshlang'ich qiymatga ega bo'lishi kerak.
  • const kalit so'zi orqali e'lon qilingan o'zgaruvchi qiymatini keyinchalik boshqa qiymatga o'zgartirib bo'lmaydi.
  • const kalit so'zining let va var dan asosiy farqi, o'zgaruvchiga yangi qiymat berib bo'lmaydi.
  • const kalit so'zi orqali o'zgaruvchini biron-bir blok ichida e'lon qiladigan bo'lsak uni faqat o'sha blok ichida ishlatishimiz mumkin bo'ladi.
3. var
var bu - Eski O'zgaruvchan qiymat.
  • var kalit so'zi orqali e'lon qilingan o'zgaruvchi qiymatini keyinchalik boshqa qiymatga ham o'zgartirib ishlatishimiz mumkin.
  • var kalit so'zining let va const dan asosiy farqi, biz var kalit so'zi orqali 2-qatorda o'zgaruvchi e'lon qilib uni 1-qatorda ishlatadigan bo'lsak bu bizga undefined qiymatini beradi. Bu esa bizga keyinchalik muammolar tug'dirishi mumkin.
    • Bu yerda xato o'rniga undefined qiymati chiqadi. Bu esa bizga muammolar tug'diradi.
